# MC-Plugin

**A modular plugin than adds many features to improve you gameplay.**

### This plugin adds some featurs like:

- Advancements
- Dimmension changes
- Mechanics
- Improved loot-tables
- Predicates
- Custom Recipes
- Custom Structures
- Block tags and worldgen
- Custom weapons
- Server protection system
- Player protecting system
- Complete multiblock structurs

# Commands

**Use /mp help to see all commands**

# Permissions

- **mcplugin.admin or mcplugin. * - grants all perms**
- **mcplugin.chat.filter.bypass - bypassing a chat filter**
- **mcplugin.command. * - plugin commands like /mp chgdim**
- **mcplugin.gmprotect.bypass - allows bypass gamemode protect it can be configured in config**
- **mcplugin.overload.logs - allows to send overload logs in chat like "[MC-Plugin] [SERVER | WARNING] MSPT=41.17767974683544" (this in console)**
- **mcplugin.packetguard.bypass - allows to bypass packet size limiter**
- **mcplugin.show.brand - allows to show server brand like "Leaf" in F3 menu if player has no perms it will show like "" in F3 menu**


# Installation

1. Download the .jar file of the plugin.
2. Put the .jar file into the plugins folder.
3. Restart the server. After the plugin has loaded, restart the server again to correctly load the data pack.

# Updating

1. Delete automatically created datapack and it will be updates and re-created by plugin.
2. Delete automatically created plugin config because without it plugin can not work correctly.

# Requirements

1. Java 21+
2. Paper/Purpur core and its forks. Do not use Spigot/Bukkit core!
